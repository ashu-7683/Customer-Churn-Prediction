# Customer Churn Prediction Analysis

## Project Overview
This project analyzes customer churn data and builds predictive models to identify customers at risk of churning.

## Dataset
- **File**: `data/customer_churn.csv`
- **Rows**: 900 customers
- **Features**: Age, Monthly Charges, Account Manager, Tenure, Number of Sites, Location, etc.
- **Target**: Churn (1 = churned, 0 = retained)
